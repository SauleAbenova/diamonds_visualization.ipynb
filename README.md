# 💎 Diamonds Data Visualization

## 📊 Dataset

This project uses the **diamonds dataset** provided by seaborn.
The dataset contains information about diamond characteristics such as price, cut, color, and depth.

---

## 📌 Objective

The goal of this project is to:

* Visualize **categorical and numerical data**
* Understand **data distribution**
* Compare different visualization techniques

---

## 📈 Visualizations

### 1. Distribution of Color

* Type: Categorical Data
* Method: Count Plot
* Description: Shows the number of diamonds for each color category

---

### 2. Distribution of Cut

* Type: Categorical Data
* Method: Count Plot
* Description: Displays the frequency of each cut quality

---

### 3. Histogram of Depth

* Type: Numerical Data
* Method: Histogram
* Description: Shows the distribution of depth values

---

### 4. Histogram of Price

* Type: Numerical Data
* Method: Histogram
* Description: Displays how diamond prices are distributed

---

### 5. Boxplot of Depth

* Type: Numerical Data
* Method: Box Plot
* Description: Shows median, spread, and outliers of depth

---

### 6. Boxplot of Price

* Type: Numerical Data
* Method: Box Plot
* Description: Displays median, variability, and outliers in price

---

## 🛠️ Tools & Libraries

* Python
* seaborn
* matplotlib

---

## 💡 Key Insights

* Categorical data is best visualized using **count plots**
* Numerical data distribution is effectively shown using **histograms**
* **Boxplots** help identify outliers and data spread

---

## 🚀 Conclusion

This project demonstrates how different visualization techniques can be used to explore and understand both categorical and numerical data effectively.
